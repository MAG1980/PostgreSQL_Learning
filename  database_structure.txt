Сущности:
    товар (id, название (модель), описание, id категории, id производителя, количество, цена, скидка, дата создания),
    категория товара (id категории, название категории, дата создания, id родительской категории),
    производитель товара (id, название, логотип, страна происхождения, дата создания),
    логотип производителя товара (id, url логотипа, размер, дата создание),
    фотография товара (id, url, размер, дата добавления),
    отзыв о товаре (id товара, id пользователя, дата создания, рейтинг, содержание отзыва, фото к отзыву),
    фотография к отзыву (id фотографии, url, размер, дата добавления),
    клиент (id пользователя, id аватара, имя, фамилия, дата рождения, электронная почта, телефон, дата
    регистрации),
    аватар клиента (id аватара, url аватара, размер, дата создания),
    избранное (id, id клиента),
    корзина (id, id клиента),
    заказ (id, id клиента, id способа оплаты, id пункта выдачи, дата оформления, статус),
    способ оплаты (id, название, дата создания),
    статус заказа (id, название, дата создания),
    пункт выдачи заказов (id, индекс, город, улица, дом, дата создания, рейтинг),

Отношения:
    Многие ко многим:
        заказ - товар;
        товар - корзина;
        товар - избранное;

    Один ко многим:
        производитель товара - товар,
        товар - фотография товара;
        товар - отзыв о товаре;
        отзыв о товаре - фотография к отзыву;
        клиент - отзыв;
        клиент - заказ;
        заказ - способ оплаты;
        заказ - статус заказа;
        заказ - способ получения;

    Один к одному:
        товар - производитель;
        производитель - логотип;
        товар - категория;
        клиент - аватар;
        клиент - избранное;
        клиент - корзина;

Таблицы связи:
    заказ_товар - (id заказа, id товара, количество товара, цена товара, скидка),
    корзина_товар - (id корзины, id товара, количество товара, цена товара, скидка),
    избранное_товар: (id избранного, id товара, id товара, дата добавления).
