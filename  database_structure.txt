Сущности:
    товар (id, название (модель), описание, id категории, id производителя, количество, цена),
    категория товара (id категории, название категории),
    логотип производителя товара (id, url логотипа),
    фотография товара (id, url, размер, дата добавления),
    отзыв о товаре (id товара, id пользователя, время создания отзыва, рейтинг, содержание отзыва, фото к отзыву),
    фотография к отзыву (id фотографии, url, размер, дата добавления),
    клиент (id пользователя, id аватара, имя, фамилия, дата рождения, электронная почта, телефон, дата
    регистрации),
    аватар клиента (id аватара, url аватара),
    избранное (id клиента, id товара),
    корзина (id, id клиента, id товара, количество товара, цена товара, скидка),
    заказ (id, id клиента, id способа оплаты, id способа получения, дата оформления, статус),
    способ оплаты (id, название),
    статус заказа (id, название),
    способ получения (id, название),
    пункт выдачи заказов (id, индекс, город, улица, дом),
    производитель товара (id, название, логотип, страна),

Отношения:
    Многие ко многим:
        товар - заказ;

    Один ко многим:
        производитель товара - товар,
        товар - категория товара;
        производитель товара - товар;
        товар - фотография товара;
        товар - отзыв о товаре;
        клиент - отзыв;
        товар - отзыв;
        отзыв - фотография к отзыву;
        клиент - избранное;
        клиент - заказ;
        клиент - отзыв;


    Один к одному:
        товар - категория;
        производитель - логотип;

        клиент - корзина;
        клиент - аватар;




Таблицы связи:
    корзина: клиент - товар (id клиента, id товара, количество товара, цена товара, скидка),
    заказ - товар (id заказа, id товара, количество товара, цена товара, скидка).
